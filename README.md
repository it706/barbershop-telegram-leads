# NordCut - барбершоп с Telegram-заявками

Учебно-коммерческий лендинг барбершопа на Next.js с рабочей отправкой заявок в Telegram.

## Задача

Создать сайт для онлайн-записи в барбершоп: клиент выбирает услугу, мастера, дату и время, а администратор получает заявку в Telegram.

## Что реализовано

- Брендовый лендинг NordCut в стиле барбершопа.
- Прайс услуг с премиальной позицией "Стрижка NordCut".
- Блок барберов с интерактивными карточками.
- Форма записи с выбором услуги, мастера, даты и времени.
- Маска телефона и проверка номера на 11 цифр.
- API endpoint `/api/lead`.
- Отправка заявки в Telegram Bot API через переменные окружения.
- Обработка ошибок для пользователя.

## Технологии

- Next.js
- React
- TypeScript
- CSS
- Telegram Bot API

## Переменные окружения

Создайте файл `.env.local`:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
```

Токен нельзя выкладывать на GitHub. Файл `.env.local` уже добавлен в `.gitignore`.

## Запуск

```bash
npm install
npm run dev
```

Локальный адрес:

```text
http://localhost:3000
```
